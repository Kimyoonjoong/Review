코딩 실습 환경 준비

코딩을 위해서는 Editor프로 그램 필요 ex) 윈도-메모장

수업에서 사용하는 Editor는 Atom(atom.io 접속하여 다운 및 설치)

설치가 끝나면 바탕화면에 web이라는 폴더를 만들기

atom에서 메뉴 -> File -> Add project folder를 선택하여 바탕화면에 만든 web 폴더 선택

atom 화면 왼쪽 Project 뷰에 생긴 web폴더를 오른쪽 클릭하여 New File 선택

*MS 워드의 확장자는 doc, PDF는 pdf, 웹브라우저에서 볼 수 있는 웹페이지는 확장자가 html이기 때문에 파일이름(상관없음).html을 붙여줘야 한다.

웹을 브라우저로 열기 위해서는 Ctrl+O 누르기, 누르면 나오는 파일선택 화면에서 바탕화면에 만든 web폴더 선택

그 후에 atom 화면 오른쪽에 아무렇게 적고 ex) hello web 저장하면 웹 브라우저에 내가 쓴 글이 나타남

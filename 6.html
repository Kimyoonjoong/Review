부모 자식과 목록(목차)

li /li:목차(list)를 만들 때 사용

ul /ul:li의 부모가 되는 코드로 li를 한번에 묶어 주는 역할을 하며 목록과 목록 사이를구분할 수 있도록 경계가 필요하기 때문에 사용(li를 사용 하면 무조건 ul를 사용해야 한다.)

ol /ol:ul과 똑 같은 코드 지만 ol은 자동으로 순서를 매겨준다. Ex) 1.html 2.css 3.java

Tip ctrl 키를 누르고 마우스 버튼을 누르면 여러곳에 커서가 생김(동일한 코드 한번에 여러줄 작성 용이)

tap키를 누르면 코드가 좀더 보기 쉽게 포함이 됨.(li가 ul코드 안으로 들어감)
